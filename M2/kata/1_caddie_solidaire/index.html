Panier Solidaire

Énoncé

Tu as un panier d’achats. Chaque item est un objet :

{ name: "bananas", price: 1.2, quantity: 3 }

Écris calculateTotal(cart) qui retourne le total :

total = somme de price * quantity

si quantity >= 5 sur un item → -10% sur cet item uniquement

retourne un nombre arrondi à 2 décimales

Exemple :

5 pommes à 2€ = 10€ → -10% → 9€
